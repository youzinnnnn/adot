문장 넘버링 예시

지문을 문장 단위로 자동 분리하고, 순서대로 번호를 붙여줍니다. '제목 포함하기'를 체크하면 문제 번호 등을 함께 넣을 수 있습니다.

✍️ 입력 예시

[제목 입력란]

2과 5번

[지문 입력란]

The robot wakes up. It looks at the sun. "Good morning!" it says.
✨ 생성 결과
[2과 5번]

(1) The robot wakes up.

(2) It looks at the sun.

(3) "Good morning!" it says.

순서 배열 예시

지문을 문장 단위로 나눈 뒤, 순서를 무작위로 섞어 순서배열 문제를 만듭니다. '제목 포함하기'를 체크하면 문제 번호 등을 함께 넣을 수 있습니다.

✍️ 입력 예시

[제목 입력란]

1강 3번

[지문 입력란]

First, gather your ingredients. Second, mix them in a bowl. Finally, bake for 20 minutes.
✨ 생성 결과
[1강 3번]
                                        ① Finally, bake for 20 minutes.

② First, gather your ingredients.

③ Second, mix them in a bowl.

정답: ② → ③ → ①

어순 배열 예시

지문을 문장 단위로 나눈 뒤, 각 문장의 단어 순서를 무작위로 섞어 어순배열 문제를 만듭니다. '해설 포함하기'를 체크하고 각 문장에 맞는 우리말 해석을 입력하면 문제에 함께 표시됩니다.

✍️ 입력 예시

[해설 입력란]

아기 곰은 눈덩이를 만들어요. 눈 오는 행복한 하루예요. 엄마 곰이 웃어요.

[지문 입력란]

Baby Bear makes a snowball. It's a happy snowy day. Mama Bear laughs.
✨ 생성 결과
(1) 아기 곰은 눈덩이를 만들어요.
[ makes / bear / snowball / baby / a ]
→

(2) 눈 오는 행복한 하루예요.
[ a / snowy / day / happy / it's ]
→
.
.

구문 배열 예시 (AI)

지문을 문장 단위로 나눈 뒤, 의미 단위(구문)로 나누고 순서를 섞어 배열 문제를 만듭니다. '해설 포함하기'를 체크하고 각 문장에 맞는 우리말 해석을 입력하면 문제에 함께 표시됩니다.

✍️ 입력 예시

[해설 입력란]

나는 공원에서 책 읽는 것을 즐긴다. 나무들이 부드럽게 바람에 흔들린다.

[지문 입력란]

I like to read books in the park. The trees sway gently in the breeze.
✨ 생성 결과
(1) 나는 공원에서 책 읽는 것을 즐긴다.
[ in / books / the park / like to read / I ]
→

(2) 나무들이 부드럽게 바람에 흔들린다.
[ sway gently / in / the breeze / The trees ]
→
